# 003 Product Detail Page Report

작성일: 2026-06-26

## 목표

선착순 한정 수량 드롭 쇼핑몰 도메인에 맞게 상품 상세 페이지를 단순 상품 정보 화면에서 긴 상세 페이지 구조로 확장했다.

요청 섹션:

- 상품 대표 사진, 상품명, 가격, 드롭 오픈 시간, 남은 수량
- 착용/무드 사진 2장
- 상품 포인트 설명
- 소재/디테일 섹션
- 사이즈 가이드 HTML 표
- 배송/교환 안내
- 추천 상품

## 컴포넌트와 섹션 정리

컴포넌트는 React에서 UI와 로직을 재사용 가능한 단위로 묶은 함수다. 예를 들어 `ProductCard`, `QuantityField`, `ProductMoodSection`이 컴포넌트다.

섹션은 화면에서 사용자가 보는 의미 있는 영역이다. 예를 들어 `착용/무드 컷`, `사이즈 가이드`, `배송/교환 안내`가 섹션이다.

둘의 관계는 1:1로 고정되지 않는다.

- 작은 컴포넌트: 버튼, 수량 입력, 상품 카드
- 섹션 단위 컴포넌트: 무드 섹션, 사이즈 가이드 섹션
- 페이지 단위 컴포넌트: 상품 상세 페이지 전체

이번 작업에서는 화면 섹션을 각각 React 컴포넌트로 나눴기 때문에, 사용자가 보는 구조와 개발자가 유지보수하는 구조가 거의 일치한다.

## 남은 리스크

- 현재 무드/디테일 이미지는 같은 카테고리 추천 상품 이미지를 재활용한다. 상품별 전용 착용 컷, 소재 컷 데이터를 백엔드가 제공하는 구조는 아직 아니다.
- 상세 설명도 카테고리 기준 템플릿이다. 실제 서비스 수준으로 가려면 상품별 상세 속성, 소재, 실측, 배송 정책을 별도 데이터로 관리하는 API 설계가 필요하다.
