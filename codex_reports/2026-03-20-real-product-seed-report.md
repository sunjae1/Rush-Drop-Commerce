# 2026-03-20 실상품 데이터 주입 보고서

## 1. 작업 목적

- 프론트 목업이 아니라 실제 DB + 현재 연결된 S3를 사용하는 상품 데이터를 채웠습니다.
- 현재 운영 중인 실카테고리 기준으로 `상의 / 하의 / 아우터`가 각각 10개씩 보이도록 맞췄습니다.

## 2. 수행 방식

- 관리자 권한이 필요한 `POST /api/items` API를 사용했습니다.
- 이미지 파일은 무료 사용 기준의 Pexels 이미지를 내려받은 뒤, 멀티파트 업로드로 S3에 저장되도록 처리했습니다.
- 즉 DB에는 새 상품이 실제로 생성되었고, 이미지도 현재 연결된 S3 버킷에 실제 업로드되었습니다.
